object structure
0x00 - id
0 - item
1 - entity / controllable
2 - door
3 - sign
4 - level parts / tile
5 - background parts
0x01 - id of id (lol)
item:
0 - gem
1 - diamond (whatever that is)
2 - heart
3 - star
4 - life
entity:
0 - enemy (red)
1 - enemy (green)
2 - floating ball
3 - box (based on red enemy)
door:
0 - finish door
1 - locked door (red key)
2 - locked door (blue key)
3 - linked warp
level parts:
0 - ground
X - spike / damage
X - turbulance
background parts:
X - fence
